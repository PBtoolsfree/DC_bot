# Support

## Where to get help

If you are having trouble deploying or using the Enterprise Discord Management Platform, please check the following resources first:

1. **[FAQ](FAQ.md)** - Answers to common questions.
2. **[Troubleshooting Guide](TROUBLESHOOTING.md)** - Solutions for common deployment and runtime errors.
3. **[Deployment Guide](docs/deployment_guide.md)** - Step-by-step instructions for Docker and Bare-metal setups.

## Bug Reports

If you believe you have found a bug in the software, please open an issue on GitHub. Include:
* Your deployment method (Docker, Bare-metal, Oracle Cloud).
* The OS and Python version you are using.
* The exact error message or stack trace.
* Steps to reproduce the bug.

## Feature Requests

We welcome feature requests! Please open an issue on GitHub using the "Feature Request" template. Keep in mind that we are currently focused on stabilizing Phase 1 features, so major new functionality may be scheduled for future releases.

## Security Vulnerabilities

Please **DO NOT** open a public issue for security vulnerabilities. Review our [Security Policy](SECURITY.md) for instructions on how to securely report vulnerabilities.
